# Elas na TI

## Sobre o projeto

O **Elas na TI** é um projeto desenvolvido com o objetivo de dar visibilidade às experiências, opiniões e desafios enfrentados por mulheres na área de Tecnologia da Informação.

A plataforma permite que usuárias compartilhem seus depoimentos sobre o mercado de TI, suas perspectivas para o futuro da área, sugestões para as novas gerações e os fatores que ainda contribuem para a baixa representatividade feminina no setor.

O projeto busca incentivar a participação de mulheres na tecnologia, promover a troca de experiências e contribuir para a construção de um ambiente mais diverso e inclusivo.

---

## Funcionalidades

- Cadastro de depoimentos.
- Exibição pública dos depoimentos cadastrados.
- Inclusão opcional de perfil do LinkedIn.
- Armazenamento dos dados em banco de dados Firebase Firestore.
- Interface simples e responsiva.

## Como executar o projeto

1. Clone ou baixe este repositório.
2. Abra o projeto em um servidor local.
3. Acesse a página inicial.
4. Navegue entre as páginas e utilize o formulário para enviar novos depoimentos.

---

## Objetivo

Promover a representatividade feminina na Tecnologia da Informação por meio do compartilhamento de experiências, incentivando novas gerações a ingressarem e permanecerem na área.
